Frequently Asked Questions

Find answers to common questions about My Nihongo Path.

Is this website free to use?

Yes, My Nihongo Path is completely free to use. It is a personal project created to help others learn Japanese.

Will you add Katakana and Kanji?

Yes! We are currently focusing on Hiragana, but we plan to add Katakana and basic Kanji sets in future updates. Stay tuned!

How do I use the practice feature?

First, browse the Hiragana sets (Set 01, Set 02, etc.) to familiarize yourself with the letters. When you're ready, click the "LET'S PRACTICE" button on any set page. This will take you to a practice session where you can test your memory.
