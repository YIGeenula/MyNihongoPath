Frequently Asked Questions

Find answers to common questions about learning with My Nihongo Path.

payments

Is this website free to use?

Yes, My Nihongo Path is completely free to use. It is a personal project created to help others learn Japanese without any cost barriers.

update

Will you add Katakana and Kanji?

Yes! We are currently focusing on Hiragana, but we plan to add Katakana and basic Kanji sets in future updates. We're constantly working on expanding our content.

school

How do I use the practice feature?

First, browse the Hiragana sets (Set 01, Set 02, etc.) to familiarize yourself with the letters. When you're ready, click the "Let's Practice" button on any set page. This will take you to a practice session where you can test your memory with interactive drills.
